Hunt | Landing Page

Sitio web desarrollado en 2021 por Sebastián utilizando Bootstrap y JavaScript Hunt es un producto desarrollado en Angular, del cual también formó parte como Desarrollador Frontend, pero no hay registro visual de ello. En este producto se encargó de trabajar el concepto de marca, crear la nomenclatura del producto, diseñar logotipo, imagen corporativa y desarrollo de Landing Page con animaciones CSS3 en todas las imágenes del home. Es un sitio estático que muestra los servicios que ofrece el producto, la posibilidad de contactar al proveedor y solicitar una demo una demo a partir del uso de un formulario. Fue desarrollado para Stormtech SL.
